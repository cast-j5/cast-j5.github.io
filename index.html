How I Learn

To learn about software, I like to reverse engineer software to understand the design, architecture and functionality. I do this by reviewing an applications code, components and behavior, using this information I can make a model of the application or system such as a class diagram to represent relationships, associations and how other various components interact with each other to make the application work. Models like this can help stakeholders understand the functionality of a application.

Immersive learning has helped me become more tenacious when learning about a new topic in computer science or programming language. Using immersive learning I get hands on experience with a new topic, and my practical skills and critical thinking also improve with hands on experience. This has helped me not be discouraged when learning a topic that seems difficult, overall I enjoy to learn about knew things in computer science with hands on experience and using this experience to create solutions for problems.
